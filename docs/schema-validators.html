Validators check to see if value matches the requirement. If it matches the validator will return true. If it doesn’t match it return false.

Schema Validators

isArray

Returns true if object is an Array.

isBoolean

Returns true if object is either true or false.

isDate

Returns true if object is a Date.

isDefined

Returns true if object is not undefined.

isEmpty

Returns true if object contains no values (no enumerable own-properties).

isFunction

Returns true if object is a Function.

isNull

Returns true if the value of object is null.

isNumber

Returns true if object is a Number (including NaN).

isNumeric

Returns true if object represents a numeric value.

isObject

Returns true if value is an Object. Note that JavaScript arrays and functions are objects, while null, (normal) strings and numbers are not.

isRegExp

Returns true if object is a RegExp.

isSchema

Returns true if object is a Cobra Schema.

isString

Returns true if object is a String.

isUndefined

Returns true if object is undefined.
