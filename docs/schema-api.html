Cobra provides several helper functions to help format data.

schemaType

Getter / Setter used to create custom SchemaTypes. If the function is undefined, the function will act as a getter. For details, view documentation “Custom schema types”.

Getter / Setter used to create custom SchemaHelpers. If the function is undefined, the function will act as a getter. For details, view documentation “Custom schema helpers”.
